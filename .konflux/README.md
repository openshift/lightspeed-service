# Konflux Hermetic Build Files

This directory contains configuration and generated requirements files for the
[Konflux](https://konflux-ci.dev/) hermetic build pipeline. These files are
referenced by the Tekton pipeline definitions in `.tekton/`.

## Configuration (checked in, manually maintained)

| File | Description |
|------|-------------|
| `profiles.toml` | Build profile configuration: RHOAI index URL, target platforms, bootstrap packages, and Tekton pipeline file paths. |
| `requirements.overrides.txt` | Version pins that steer `uv pip compile` toward versions available on the RHOAI index. |
| `requirements.hermetic.txt` | Build tool pins (`uv`, `uv-build`) prefetched by Cachi2 for the hermetic build. |
| `pypi_wheel_only.txt` | Packages that only have wheel distributions on PyPI (no sdist). Listed here to suppress auto-promotion warnings. |
| `rpms.in.yaml` | RPM package list and repo configuration for `rpm-lockfile-prototype`. |
| `redhat.repo` | Red Hat repository definitions extracted from the base image, used by RPM lock generation. |

## Generated (regenerated by `make konflux-requirements`)

| File | Description |
|------|-------------|
| `requirements.hashes.wheel.txt` | RHOAI wheel packages with SHA-256 hashes. |
| `requirements.hashes.source.txt` | PyPI source (sdist) packages with SHA-256 hashes. |
| `requirements.hashes.wheel.pypi.txt` | PyPI wheel packages with SHA-256 hashes (packages without sdist on PyPI). |
| `requirements-build.txt` | Build-time dependencies for source packages (generated by `pybuild-deps`). |

## Generated (regenerated by `make konflux-rpm-lock`)

| File | Description |
|------|-------------|
| `rpms.lock.yaml` | Pinned RPM package versions for the hermetic build. |

## Regenerating

See [docs/update-requirements.md](../docs/update-requirements.md) for instructions.

```bash
make konflux-requirements
```
